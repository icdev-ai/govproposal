# Key FAR/DFARS Clauses for DoD IT Services Proposals

## FAR (Federal Acquisition Regulation)

### FAR 3.104 — Procurement Integrity Act
- Prohibits disclosure of contractor bid/proposal information and source selection information
- Applies to all personnel involved in the acquisition process
- Violations can result in contract cancellation, fines, and imprisonment
- **GovProposal Impact:** All proposal content is PROPIN — competition-sensitive

### FAR 15.3 — Source Selection
- Defines the source selection process for competitive acquisitions
- Evaluation factors must be stated in the solicitation (Section M)
- Price/cost must be evaluated in every source selection
- Best value = trade-off between price and non-price factors
- **GovProposal Impact:** Proposal structure must align with stated evaluation criteria

### FAR 52.204-21 — Basic Safeguarding of Covered Contractor Information Systems
- 15 basic security requirements for contractor information systems
- Applies when contractor processes, stores, or transmits Federal contract information
- **GovProposal Impact:** Baseline security for all proposal data handling

### FAR 52.215-1 — Instructions to Offerors
- Defines proposal submission requirements
- Late proposals generally not considered
- Amendments must be acknowledged
- **GovProposal Impact:** Submission packager must comply with format and deadline requirements

## DFARS (Defense Federal Acquisition Regulation Supplement)

### DFARS 204.73 / 252.204-7012 — Safeguarding Covered Defense Information
- Requires NIST SP 800-171 compliance for CUI on contractor systems
- Mandatory cyber incident reporting within 72 hours
- Cloud services must meet FedRAMP Moderate or DoD equivalency
- **GovProposal Impact:** Systems handling CUI must be NIST 800-171 compliant

### DFARS 252.204-7019/7020/7021 — CMMC Requirements
- Contractors must achieve appropriate CMMC certification level
- Self-assessment scores recorded in SPRS
- Third-party assessment for Level 2+
- **GovProposal Impact:** Proposals must address CMMC readiness

### DFARS 252.239-7010 — Cloud Computing Services
- DoD cloud services must meet IL requirements
- Data must remain within US or US territories
- CSP must hold FedRAMP authorization at appropriate level
- **GovProposal Impact:** Cloud architecture proposals must specify IL level and FedRAMP status

## Applicable Executive Orders

### EO 13526 — Classified National Security Information
- Section 1.7(e) — Compilation/mosaic effect: individually unclassified items may be classified when aggregated
- **GovProposal Impact:** Primary driver for the CAG system

### DoDM 5200.01 — Information Security Program
- Implements classification marking, handling, and safeguarding procedures
- Defines Security Classification Guides (SCGs) for programs
- **GovProposal Impact:** SCG parser imports program-specific classification rules into CAG
